# 免費版與盤勢 Pro

盤勢的免費版每天都有可用內容，不是限時試用。訂閱購買的是不限檔數、研究深度與無廣告體驗，不是買賣答案。

| 體驗 | 免費版 | 盤勢 Pro |
| --- | --- | --- |
| 今日五盤 | 每日 5 檔完整使用，不扣查詢額度 | 同左 |
| 額外股票查詢 | 每天 3 檔不同股票；同一檔重查不重扣 | 不限檔數 |
| 公司盤與基本解讀 | 在上述檔數內完整使用 | 完整使用 |

| 完整歷史檔案 | 可自願觀看 30–60 秒獎勵式影片，解鎖單份檔案 24 小時 | 持續開啟 |
| 觀察簿與回看提醒 | 完整使用 | 完整使用 |

| 首次訂閱 | — | 符合資格者 7 天免費，之後按月續訂 |

額度依臺北日期計算，午夜重置。取消試用或訂閱到期後，App 回到每日五盤加 3 檔額外查詢；本機觀察簿不刪除。

## 查詢紀錄

- 公司盤與日期問盤會記錄股票代號、問題日期、命盤基準、觀察期與請求時間，供額度核對與重複問題統計。
- 安裝識別碼進入資料庫前先轉成不可逆雜湊；不保存觀察簿文字，也不保存付款資料。
- 個別請求紀錄保留 180 天。問題目錄只保留問題本身、首次與最近被問時間及累計次數，不含安裝識別碼。

## 廣告節奏

- 啟動、搜尋輸入、命盤閱讀途中與錯誤畫面不插廣告。
- 一般插頁只在一次研究完成後考慮出現，最多每三次完成動作一次。
- 30–60 秒影片只用於使用者主動選擇的 24 小時深度檔案解鎖。
- 每個廣告都必須標示為廣告，提供可見的關閉方式與不當內容回報入口。
- 預設採內容情境式投放，不使用公司查詢、收藏或觀察簿建立跨 App 側寫。

## 不拿來變現的東西

- 不販售個人化買進、賣出或持有結論。
- 不把「勝率」、目標價、進出場點、停損、槓桿或部位配置放進任何方案。
- 不因為使用者付費，就把小樣本、缺資料或相互矛盾的案例寫成結論。
- 不以倒數、假折扣或鎖住既有筆記逼迫訂閱。
