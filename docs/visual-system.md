# 盤勢視覺與動效系統

_2026-07-11 版_

## 設計讀法

盤勢是一件「當代觀象儀器」，不是算命攤，也不是交易終端。首頁服務對玄學好奇、同時在意資料來源與限制的人。

- `DESIGN_VARIANCE: 8`：非對稱紙面與深色觀象器物並置。
- `MOTION_INTENSITY: 8`：互動集中在有敘事意義的時刻，不讓整頁持續漂浮。
- `VISUAL_DENSITY: 5`：品牌首頁留白，研究工作台維持可掃讀密度。

玄學色彩由構圖、材質、命名與儀式感形成。資料、錯誤、風險與免責仍使用直接語言。

## 主視覺

首頁使用 `public/images/panshi-celestial-market.webp`。這是為盤勢生成的無字主視覺：黑鋼與氧化黃銅天體儀、骨白刻度、礦物藍歷史價格軌跡與單一朱砂印記。

圖片不承載標題、按鈕或說明。所有重要文字保留為 HTML，確保可存取、可翻譯、可搜尋，也避免生成文字失真。

互動鏡頭有三個語意位置：

1. 命盤基準：看日期如何成為起盤依據。
2. 價格軌跡：看記錄，不暗示下一步方向。
3. 觀測印記：先留下問題，再回頭驗證。

## 字型

- 品牌與大型標題：Chiron Sung HK Variable（昭源宋體），SIL Open Font License 1.1。它保留東亞印刷骨架，也有金融出版需要的可信度，只用於品牌與敘事層級。
- UI 與長文：Noto Sans TC、PingFang TC、Microsoft JhengHei 與系統無襯線 fallback。
- 價格、日期、ticker 與英文標籤：IBM Plex Mono，SIL Open Font License 1.1。

品牌宋體依實際介面字形裁成單一 WOFF2 子集，保留 `wght` 與 `PADG` 可變軸；OFL 授權副本與字檔一起保存。等寬字只載入 Latin 400 與 600。不要把宋體套進表格、表單或密集資料；新增展示文案時也要同步重建字形子集。

## 動效邊界

- 主視覺入場一次，游標只造成小幅傾斜與視差。
- 三個鏡頭切換使用 spring，改變視點而不是製造裝飾。
- 工作台區塊可用瀏覽器 scroll-driven animation 進場；不支援時直接呈現。
- loading、選取事件、收藏觀測等回饋各發生一次，不做無限循環。
- reduced motion、觸控與粗略指標裝置停用視差並直接顯示終態。

## 品質門檻

- Hero 圖片保留固定尺寸並優先載入。
- 動畫只改 `transform` 與 `opacity`。
- 不使用紫色 mesh、霓虹外光、假 dashboard、股價預測箭頭、粒子爆炸或自訂游標。
- 每次視覺調整都要檢查桌面、窄螢幕、觸控、鍵盤與 reduced motion。
- 目標：LCP < 2.5 秒、INP < 200 毫秒、CLS < 0.1。
